# unit-converter-ai-mcp

MCP server for unit conversion tools.

## Tools

- **convert_length** — Convert between length units (metric and imperial)
- **convert_weight** — Convert between weight/mass units
- **convert_temperature** — Convert between temperature scales
- **convert_currency_data** — Currency conversion with static reference rates

## Usage

```bash
pip install mcp
python server.py
```

## Rate Limits

50 calls/day per tool (free tier).
